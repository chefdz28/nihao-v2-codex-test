# 🚀 NiHao V3.17 — دليل النشر الكامل (3 أنظمة مدمجة)

هذا الإصدار يدمج: **نظام الإحالات** + **المعلم الذكي بـCohere** + **تدريب النطق بـGroq**.
كل نظام يشتغل مستقلاً — لو ما فعّلت مفاتيح AI، الإحالات تشتغل والموقع طبيعي 100%.

## الخطوة 1: شغّل الـ3 migrations في Supabase (بالترتيب)
افتح Supabase → SQL Editor وشغّل واحداً واحداً:
1. `supabase/migrations/20260619_referral_system.sql` (الإحالات — رموز + مكافآت)
2. `supabase/migrations/20260620_ai_teacher_usage.sql` (حد المعلم الذكي اليومي)
3. `supabase/migrations/20260620_pronunciation_usage.sql` (حد النطق اليومي)
كلها idempotent — إعادة تشغيلها آمنة.

## الخطوة 2: انشر الـEdge Functions (2)
```bash
supabase functions deploy ai-teacher-cohere --project-ref <REF>
supabase functions deploy pronunciation-check --project-ref <REF>
```
أو انسخ الملفات من Dashboard → Edge Functions:
- `supabase/functions/ai-teacher-cohere/index.ts`
- `supabase/functions/pronunciation-check/index.ts`

## الخطوة 3: أضف المفاتيح (Secrets)
Supabase → Edge Functions → Secrets:
| السر | من وين | ملاحظة |
|---|---|---|
| `GROQ_API_KEY` | console.groq.com (مجاني بلا بطاقة) | للنطق |
| `COHERE_API_KEY` | cohere.com → API Keys (Production) | للمعلم الذكي |

⚠️ تقدر تبدأ بـGroq فقط (مجاني) وتأجّل Cohere — المعلم يرجع للروابط المحلية بأمان.

## الخطوة 4: انشر الموقع (نفس طريقتك المعتادة)
- الحزمة: `nihao-v3.17-growth-and-ai.zip` (الإصدار 3.17.0)
- مبنية على آخر main (da90f20 / V3.15.1) → آمنة مع rsync --delete
- نفس أوامر النشر: unzip → cp .env → npm install → build → backup → نسخ dist → git sync

## الخطوة 5: اختبر

## ماذا لو ما فعّلت المفاتيح؟
| النظام | بدون مفتاح |
|---|---|
| الإحالات | ✅ يشتغل كاملاً (لا يحتاج مفاتيح) |
| المعلم الذكي | يرجع للروابط المحلية (زي قبل) |
